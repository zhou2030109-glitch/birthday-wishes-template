# 生日祝福模板

一个静态生日祝福页面模板，使用 Three.js 渲染蛋糕、蜡烛和许愿动画。

## 自定义

- 将 `{{RECIPIENT_NAME}}` 替换为收件人姓名。
- 将 `{{BIRTHDAY_DATE}}` 替换为展示日期，例如 `DD MON` 或 `YYYY年MM月DD日`。
- 如果需要背景音乐，把音频文件放到 `music/bgm.mp3`。
- 可直接部署到 GitHub Pages 或任意静态站点托管服务。

## 文件

- `index.html`: 页面结构和模板文本。
- `main.js`: Three.js 动画、交互和显示日期。
- `style.css`: 页面样式和响应式布局。
